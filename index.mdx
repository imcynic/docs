# Welcome to 04Scapes Engine

The 04Scapes Engine is a high-performance, TypeScript-based MMORPG server implementation designed to recreate the classic RuneScape experience from May 18, 2004. This documentation provides comprehensive guidance for developers working with the engine.

## What is 04Scapes Engine?

04Scapes is a complete game server implementation featuring:

- **Full Game Engine**: Complete world simulation with tick-based processing
- **Entity Management**: Players, NPCs, objects, and location handling
- **RuneScript System**: Powerful scripting engine for game content
- **Network Protocol**: Client-server communication with packet handling
- **Database Integration**: MySQL/SQLite support for persistence
- **Cache System**: Efficient data packing and loading
- **Multi-world Support**: Scalable architecture for multiple game worlds

## Technology Stack

- **Runtime**: [Bun](https://bun.sh/) - Fast JavaScript runtime
- **Language**: TypeScript - Type-safe development
- **Database**: MySQL/SQLite with Prisma ORM
- **Networking**: TCP/WebSocket support
- **Testing**: Built-in test framework
- **Build Tools**: Custom packing and compilation tools

## System Requirements

- Node.js 22+ or Bun runtime
- Java 17+ (for client tools)
- 4GB RAM minimum
- Windows, macOS, or Linux
